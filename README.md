    shell works
## Getting  Started

    1. Clone the repo:

            - $ git clone https://github.com/techmate-dot/simple_shell.git

    2. change directory  to repo

    3.  Compile the code using:

           - $ gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh

    4. Run the code:
            - $./hsh
